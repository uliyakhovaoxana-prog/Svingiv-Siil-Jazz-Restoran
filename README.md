# Svingiv-Siil-Jazz-Restoran
Svingiv Siil on New Orleansi vaimust kantud džässirestoran Tallinna südames.  See on paik, kus kohtuvad sametine ööpimedus, särav vask ja meisterlik kulinaaria.  Meie juures on muusika midagi enamat kui heli — see on elustiil, mis ühendab  endas luksusliku elegantsi ja koduse hubasuse, kus iga noot paitab hinge.
